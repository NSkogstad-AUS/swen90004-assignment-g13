## Statement of AI Tool Use

This group used AI-assisted tools during the development of this assignment.  The following
describes the nature of that use.

### Tools Used

- **Claude (Anthropic)** — used via the Claude Code CLI.

### Tasks Where AI Assistance Was Used

| Task                         | Description of AI involvement                                   |
|------------------------------|-----------------------------------------------------------------|
| Repository structure         | AI suggested the initial directory layout and file names.       |
| Code generation              | AI generated initial drafts of Python source files.             |
| Documentation drafting       | AI produced initial drafts of README, MODEL_DESIGN, and         |
|                              | EXPERIMENT_PLAN.  These were reviewed and edited by the group.  |
| Debugging                    | AI assisted in identifying and resolving logic errors during     |
|                              | development.                                                    |
| Test case generation         | AI generated initial unit test skeletons.                       |
| Summarisation                | AI was used to summarise requirements from the assignment brief. |

### What AI Did Not Do

- AI did not run the NetLogo model or produce NetLogo output values.
- AI did not make final decisions about the model design or extension choice.
- AI did not write the final report narrative.
- AI did not review or interpret experimental results.
